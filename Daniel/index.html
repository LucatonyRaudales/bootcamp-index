¿Qué es GitHub?

GitHub no es solo un lugar para guardar código; es la plataforma de desarrollo colaborativo más grande del mundo. Piensa en él como una red social para programadores donde puedes alojar tus proyectos, trabajar con otros y construir software increíble.

¿Qué Aprenderás en este Bootcamp?

Dominarás el flujo de trabajo esencial de Git y GitHub:

  • Repositorios (Repos): Dónde vive tu proyecto.
  • Commits: Guardar "puntos de control" de tu progreso.
  • Branches (Ramas): Trabajar en nuevas funciones de forma segura sin afectar el proyecto principal.
  • Pull Requests (PRs): Proponer tus cambios para que sean revisados e integrados.
  • Merges: Combinar tus ramas de vuelta al proyecto principal.
  • Colaboración: Trabajar en equipo, resolver conflictos y revisar código.
